
# Telegram Video Downloader Bot (with Channel Subscription Gate)

Бот позволяет пользователю прислать ссылку на видео (YouTube / TikTok / Instagram) и получить файл в ответ, **только если пользователь подписан** на указанный канал.

> ⚠️ Юридически значимо: вы обязуетесь использовать бота **только для скачивания собственных материалов, материалов с явным разрешением правообладателя, или материалов в общественном достоянии**. Нарушение условий использования площадок (YouTube/Instagram/TikTok) и авторских прав может быть незаконным. Настоятельно рекомендуется получить разрешение владельца контента.

## Быстрый старт
1. Установите Python 3.10+
2. Создайте venv и активируйте его:
   python -m venv .venv
   source .venv/bin/activate
3. Установите зависимости:
   pip install -r requirements.txt
4. Создайте .env по примеру .env.example и заполните переменные.
5. Запустите: python app.py
